# Release War Room

Key: release-war-room

## Task
Drive a staged release workflow: assess failures, propose fixes, run checks, and produce a release decision with rollback notes.

## Expected Outcome
Return the decision record, evidence from checks, and a rollback or follow-up plan when the release should not proceed.

## Metadata
Resolution: campaign
Interaction: terminal
Evaluator: hybrid
Difficulty: high
Tags: release, debugging, handoff
Requires Isolation: yes
Requires Network: no
